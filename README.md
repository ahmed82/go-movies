# Go-Movies
CRUD operations

```shell
go mod init github.com/ahmed82/go-movies
go get "github.com/gorilla/mux"

```

## Run
```
go build
go run main.go
```

API call

```
http://localhost:8000/movies
```
## Curl
```
curl --location --request GET 'http://localhost:8000/movies'
```
